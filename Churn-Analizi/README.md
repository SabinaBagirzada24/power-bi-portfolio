# Churn-Analizi (Power BI)

## Layihənin məqsədi
Bu hesabatın məqsədi müştəri axınına dair əsas rəqəmləri təqdim etməkdir.
Məqsəd, ümumi churn səviyyəsini tez və aydın şəkildə göstərməkdir.

## İstifadə olunan alətlər
- Power BI
- Power Query
- DAX

## Dataset haqqında
Dataset **simulyasiya edilmiş satış və əməliyyat məlumatlarından** ibarətdir.
Real şirkət məlumatları istifadə edilməmişdir.
