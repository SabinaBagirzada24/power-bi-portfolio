# Şəhərlər üzrə Hesabat (Power BI)

## Layihənin məqsədi
Bu hesabat satış və əməliyyat məlumatlarını şəhərlər üzrə təhlil etmək,
regional performansı müqayisə etmək və qərarverməni dəstəkləyən analitik baxış təqdim etmək üçün hazırlanmışdır.

## Əsas göstəricilər
- Şəhərlər üzrə satış həcmi
- Regionlar üzrə performans
- İl, ay, kuryer və menecer üzrə filtrasiya

## İstifadə olunan vizuallar
- Xəritə (Map)
- Slicer-lər
- 
## İstifadə olunan alətlər
- Power BI
- Power Query
- DAX

## Dataset haqqında
Dataset analitik məqsədlər üçün hazırlanmış **simulyasiya edilmiş satış məlumatlarından** ibarətdir.
Real şirkət məlumatları istifadə edilməmişdir.
