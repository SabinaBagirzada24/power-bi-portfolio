# Handex – Satış Hesabatı (Power BI)

## Layihə haqqında
Layihə **tədris məqsədilə** kurs çərçivəsində təqdim edilmiş **xarici dataset** üzərində qurulmuşdur 
və real şirkət məlumatlarını əks etdirmir.
Məqsəd satış məlumatlarının analizi, vizuallaşdırılması və
interaktiv hesabatların qurulması bacarıqlarını inkişaf etdirməkdir.

## Hesabat səhifələri
- Mündəricat (naviqasiya səhifəsi)
- Ümumi Satışların İcmalı
- Regionlara Görə Satışlar
- Məhsul Analizi
- Müştəri və Satıcı Analizi
- Satış Analizi

## Əsas imkanlar
- Drill-through və səhifələrarası keçid
- Dinamik filtrlər
- KPI və performans göstəriciləri
- Vizual müqayisələr

## İstifadə olunan texnologiyalar
- Power BI
- DAX
- Data Modeling
- Interactive Navigation
