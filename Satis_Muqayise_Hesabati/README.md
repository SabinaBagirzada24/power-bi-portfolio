# Satış Müqayisə Hesabatı (Power BI)

## Layihənin məqsədi
Bu hesabat satış göstəricilərinin illər və aylar üzrə müqayisəsini,
artım faizlərinin analizini və ən yüksək satış edən şəhərlərin müəyyən edilməsini təmin edir.

## Əsas göstəricilər
- Ümumi satış məbləği
- Keçən il ilə müqayisə
- Aylar üzrə satış dinamikası
- Satış artım faizi
- Top-10 şəhər üzrə satış

## Filtrlər
- İl
- Ay
- Menecer
- Kuryer

## İstifadə olunan alətlər
- Power BI
- Power Query 
- DAX
  
## Dataset haqqında
Dataset analitik məqsədlər üçün hazırlanmış **simulyasiya edilmiş satış məlumatlarından** ibarətdir.
Real şirkət məlumatları istifadə edilməmişdir.
