# Satış Hesabatı (Power BI)

## Layihənin məqsədi
Bu layihənin məqsədi satış məlumatlarını təhlil etmək, əsas performans göstəricilərini vizuallaşdırmaq və qərarvermə prosesini dəstəkləyən analitik baxış təqdim etməkdir.

## Əsas göstəricilər
- Ümumi satış 
- Məhsullar üzrə satış payı  
- Bölgə üzrə satış performansı   

## İstifadə olunan alətlər
- Power BI  
- Power Query  
- DAX  

## Dataset haqqında
Dataset analitik məqsədlər üçün hazırlanmış **simulyasiya edilmiş satış məlumatlarından** ibarətdir. Real şirkət məlumatı istifadə edilməmişdir.
