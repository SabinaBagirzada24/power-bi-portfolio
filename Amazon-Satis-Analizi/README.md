# Amazon Satış Analizi (Power BI)

Bu layihə Amazon satış datası əsasında hazırlanmış analitik dashboard-dur. 
Layihə **tədris məqsədilə** kurs çərçivəsində təqdim edilmiş **xarici dataset** üzərində qurulmuşdur və real şirkət məlumatı deyil.

## Layihənin məqsədi
- Satış performansını analiz etmək
- Regionlar üzrə satışları müqayisə etmək
- Stok vəziyyətini izləmək
- Müştəri churn (itki) analizini aparmaq

## Dashboard bölmələri
- **Xəritə Analizi** – region və ölkələr üzrə satışlar
- **Satış Analizi** – zaman üzrə satış dinamikası
- **Stok Analizi** – məhsul və stok vəziyyəti
- **Churn Analizi** – müştəri itkisi göstəriciləri

## İstifadə olunan alətlər
- Power BI
- Power Query
- DAX
